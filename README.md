# Simple-Liquid-Container-Monitoring-Device
## Abstract
This system takes STM32F103C8T6 as the core controller, and designs and implements a multifunctional monitoring device for liquid containers. The system integrates an ultrasonic liquid level sensor, an HX711 weight sensor, a DS18B20 temperature sensor and a self-made conductivity sensor. A rotary encoder is used to set the target liquid level and overheating threshold, and a relay is adopted to control the water pump for automatic water injection and overheating cooling. The system can accurately distinguish four kinds of liquids, including pure water, salt water, milk and white vinegar. The overheating threshold can be manually set; when the liquid is overheated, acousto-optic alarm is triggered automatically and the water pump is activated to inject water for cooling, which meets all requirements of the competition. The design features low cost, fast response and strong practicability.

本系统以STM32F103C8T6为核心控制器，设计并实现了一款多功能液体容器监控装置。系统集成超声波液位传感器、HX711重量传感器、DS18B20温度传感器及自制电导率传感器，通过旋转编码器设定目标液位和过热阈值，采用继电器控制水泵实现自动注水及过热降温。能准确区分纯净水、盐水、牛奶、白醋四种液体；可手动设置过热阈值，液体过热时自动发出声光报警并启动水泵注水降温，满足竞赛各项要求。本设计成本低、响应快，实用性强。
